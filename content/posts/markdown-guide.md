# 자주 쓰는 마크다운 문법

마크다운은 간단한 기호로 글의 구조를 표현합니다. 자주 쓰는 것들만 모았습니다.

## 강조

- *기울임* 은 별표 하나로 감쌉니다.
- **굵게** 는 별표 두 개로 감쌉니다.
- `인라인 코드` 는 백틱으로 감쌉니다.

## 목록

순서 없는 목록:

- 사과
- 바나나
- 체리

순서 있는 목록:

1. 첫째
2. 둘째
3. 셋째

## 링크와 인용

[Next.js 공식 문서](https://nextjs.org)로 이동할 수 있습니다.

> 인용문은 이렇게 표시됩니다.

## 코드 블록

```js
function greet(name) {
  return `안녕하세요, ${name}님!`;
}
```

## 표 (GFM)

| 문법 | 예시 | 결과 |
| --- | --- | --- |
| 굵게 | `**굵게**` | **굵게** |
| 코드 | `` `code` `` | `code` |

이 정도만 알아도 블로그 글을 쓰는 데 충분합니다.
